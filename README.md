# Crime-Data-Analysis
 Crime Data Analysis Dashboard (Power BI)
 
Project Overview

This project presents an interactive Crime Data Analysis Dashboard built entirely using Power BI.
The dashboard analyzes historical crime data to identify trends, patterns, and risk areas across time, location, crime types, victim demographics, and crime outcomes, enabling data-driven insights for public safety and decision-making.

Project Objective

The objective of this project is to:
1. Understand how crime patterns change over time
2. Identify the most common crime types
3. Analyze crime distribution by hour and location
4. Study victim demographics such as age, gender, and descent
5. Evaluate crime outcomes such as incidents and arrests


Tools Used

Power BI
1. Data cleaning & transformation (Power Query)
2. Data modeling
3. DAX measures
4. Interactive dashboard creation


Visualizations Used

KPI Cards - Total Crimes, Violent Crimes, Property Crimes

Line Chart - Total Crimes by Year

Bar Charts - Total Crimes by Type
             Total Crimes by Hour
             Total Crimes by Victim Descent

Donut Chart - Total Crimes by Victim Gender

Table - Total Crimes by Victim Age

Map Visualization - Crime distribution by location

Decomposition Tree - Crime Status breakdown (Incident, Arrested Adult, Other Arrest)

Slicers - Crime Type, Year


Key Insights

1. Over 1 million crime records analyzed
2. Property crimes form the largest share compared to violent crimes
3. Crime trends show a peak during recent years, followed by a decline
4. Evening and late-night hours have the highest crime occurrences
5. Vehicle-related crimes, burglary, and battery are among the most frequent
6. Majority of cases are reported as incidents, with fewer resulting in arrests
7. Victims in the late 20s to mid-30s age group are most affected
